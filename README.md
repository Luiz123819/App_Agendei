# Agendei

![Status](https://img.shields.io/badge/Status-Em%20Desenvolvimento-orange?style=for-the-badge&logo=github)

O **Agendei** é um aplicativo híbrido de agendamento de consultas médicas, desenvolvido para otimizar a conexão entre pacientes e profissionais de saúde. O projeto utiliza tecnologias web modernas integradas ao **Apache Cordova**, proporcionando uma experiência nativa em dispositivos móveis com a flexibilidade e escalabilidade de um sistema web.

> [!IMPORTANT]
> **Aviso:** Este projeto encontra-se em fase ativa de desenvolvimento. O foco atual é a finalização da integração entre o container Cordova e o backend em PHP.

## 🚀 Funcionalidades

- **Agendamento Prático:** Interface intuitiva para marcação de consultas filtradas por especialidade.
- **App Híbrido:** Experiência mobile nativa via Cordova, permitindo a instalação direta no smartphone.
- **PWA Ready:** Suporte a *Service Workers* para garantir carregamento rápido e funcionalidades offline básicas.
- **Autenticação Segura:** Sistema de login robusto para pacientes e painel para gestão de agendamentos.
- **Design Responsivo:** Layout totalmente adaptável para tablets e celulares, desenvolvido com Bootstrap.

## 🛠️ Tecnologias e Ferramentas

### **Mobile & Interface**
- **Apache Cordova:** Containerização para distribuição multiplataforma (Android/iOS).
- **HTML5, CSS3 & JavaScript:** Base da estrutura e comportamento dinâmico da aplicação.
- **Bootstrap:** Framework CSS para componentes de UI modernos e responsividade.
- **PWA (Web App Manifest):** Para integração profunda e instalação nativa no sistema operacional.

### **Backend & Infraestrutura**
- **PHP:** Processamento de requisições e implementação da lógica de negócios de agendamento.
- **MySQL:** Banco de dados relacional para persistência de informações de médicos, pacientes e consultas.
